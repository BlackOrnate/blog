# 7. feature decomposition

- The **eigenvector** of the square array $A$: multiplied by $A$ is equivalent to the non-zero vector $v$ that scales this vector:
  $$
  Av = \lambda v
  $$.
  where the scalar $λ$ is called the corresponding **eigenvalue** of this eigenvector

  - **left eigenvector**.
    $$
    v^TA = \lambda v^T
    $$

  - **right eigenvector** **right eigenvector
    $$
    Av = \lambda v
    $$

- **eigendecomposition**: decomposition of a matrix into a set of eigenvectors and eigenvalues.

  - The matrix $A$ has $n$ linearly independent eigenvectors $V = [v^{(1)}, ... , v^{(n)}]$, corresponding to $n$ eigenvalues $\lambda = [\lambda_1, ... , \lambda_n]$.

  - Eigenfactorization of the matrix $A$_
    $$
    A = Vdiag(\lambda)V^{-1}
    $$

  - In the case where $A$ is a real symmetric matrix, $V$ is an orthogonal matrix

    - $A$ viewed as a space along the direction $v^{(i)}$ extended $\lambda_i$ times



- **positive definite**: a matrix where all eigenvalues are positive.
  $$
  x^TAx = 0 \Rightarrow x = 0
  $$.

- **positive semidefinite**: a matrix in which all eigenvalues are non-negative
  $$
  \forall x, \hspace{0.1cm} x^TAx \geq 0
  $$

- **negative definite**: a matrix in which all the eigenvalues are negative.

- **negative semidefinite**: a matrix in which all eigenvalues are nonpositive.



# 8. singular value decomposition

- **Singular Value Decomposition (SVD)**: decomposes a matrix into **singular vector** and **singular value**.

- Every real matrix has a singular value decomposition, but not always an eigendecomposition

  - Non-square matrices do not have an eigendecomposition, so we can only use the singular value decomposition.

- Singular value decomposition of the matrix $A$.
  $$
  A = UDV^T
  $$
  where $A$ is a $m*n$ matrix, $U$ is a $m*m$ matrix, $D$ is a $m*n$ matrix, $V$ is a $n*n$ matrix

  - $U$ and $V$ are both orthogonal matrices, and $D$ is a diagonal matrix

  - The elements on the diagonal of $D$ are called the **singular values** of the matrix $A$.
  - The column vector of $U$ is called the **left singular vector**.
    - The eigenvectors of $AA^T$_ are called
  - The column vector of $V$ is called the right singular vector
    - Eigenvectors of $A^TA$.
  - Non-zero singular values of $A$: square root of the eigenvalues of $AA^T$ or $A^TA$.



# 9. Moore-Penrose pseudoinverse

- **Moore-Penrose pseudoinverse**

- Definition of the pseudoinverse of the matrix $A(m*n)$
  $$
  A^+ = \lim\limits_{\alpha \rightarrow 0} (A^TA + \alpha I)^{-1}A^T
  $$

- The actual algorithm for calculating the pseudoinverse is not based on this definition, but instead uses the following equation
  $$
  A^+ = VD^+U^T
  $$

  - where the matrices $U$, $D$ and $V$ are the matrices obtained by the singular value decomposition of the matrix $A$
  - $D^+$: non-zero elements are transposed after taking the inverse of the matrix.

- When $m \leqslant n$, solving linear equations using pseudo-inverses is one of many possible solutions

  - $x=A^+y$ is the smallest $||x||_2$ of all possible solutions to the equation

- When $m > n$, there may be no solution

  - The $x$ obtained by pseudo-inversion minimizes the $Ax$ and the $y$ of the $||Ax - y||_2$



# 10. Trace operations

- Trace operations: sum of diagonal elements of a matrix
  $$
  Tr(A) = \sum_{i}A_{i,i}
  $$

- Another way to describe the Frobenius paradigm of a matrix
  $$
  ||A||_F = \sqrt{Tr(AA^T)} 
  $$

- The trace operation is invariant under the transpose operation
  $$
  Tr(A) = Tr(A^T)
  $$

- The trace of a square matrix obtained by multiplying several matrices is the same as the trace obtained by multiplying the last of these matrices after moving the last to the top
  $$
  Tr(ABC) = Tr(CAB) = Tr(BCA)
  $$



# 11. rows and columns

- Row equation: a function that maps the square array $A$ to the real numbers
  - Denoted as $det(A)$.
  - Equals the product of the eigenvalues of the matrix
- The absolute value of the determinant can be used to measure how much a matrix expands or shrinks as a result of matrix multiplication.
- If the determinant is $0$, then the space is completely shrunk along at least one dimension, so that it loses all of its volume.
- If the determinant is $1$, then the transformation leaves the volume of space unchanged
