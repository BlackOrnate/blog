# 4. linear correlation and generating subspaces

- If the inverse matrix $A^{-1}$ exists, then $Ax = b$ must exist with exactly one solution for every vector $b$
- However, for a system of equations, it is possible that for some values of the vector $b$, no solution exists, or that infinitely many solutions exist
- It is unlikely that there is more than one solution but less than an infinite number of solutions.
  - Because if $x$ and $y$ are both solutions to some system of equations, then $z = \alpha x + (1 - \alpha)y$ is also a solution to the equations



- **linear combination
  $$
  Ax = \begin{bmatrix} {A_{1,1}x_1 + A_{1, 2}x_2 + ... A_{1, n}x_n} \\ {A_{2,1}x_1 + A_{2, 2}x_2 + ... A_{2, n}x_n } \\ . \\ . \\ . \\ {A_{m,1}x_1 + A_{m, 2}x_2 + ... A_{m, n}x_n} \end{bmatrix} = x_1 \begin{bmatrix} A_{1,1} \\ A_{2,1}\\ . \\ . \\ . \\ A_{m,1}  \end{bmatrix} + x_2 \begin{bmatrix} A_{1,2} \\ A_{2,2}\\ . \\ . \\ . \\ A_{m,2}  \end{bmatrix} + ... x_n \begin{bmatrix} A_{1,n} \\ A_{2,n}\\ . \\ . \\ . \\ A_{m,n}  \end{bmatrix} = \sum_{i=1}^{n} x_iA_{:, i}
  $$



- The **span** of a set of vectors: the set of points that can be reached by a linear combination of the original vectors.
  - i.e. $x$.
- Determining whether $Ax=b$ has a solution is equivalent to determining whether the vector $b$ is in the generating subspace of the column vectors of $A$
  - This particular generating subspace is called the **column space** of $A$ or the **range** of $A$.
  - i.e., the set consisting of $x$.



- **linear dependence**: any vector in a set of vectors **can** be represented as a linear combination of other vectors
- **linearly indepent**: any vector in a set of vectors **cannot** be represented as a linear combination of the other vectors



- For a matrix to be invertible, it is necessary to ensure that $Ax=b$ has at most one solution for each $b$.
  - The matrix must be a **square**.

# 7. feature decomposition

- The **eigenvector** of the square array $A$: multiplication with $A$ is equivalent to the non-zero vector $v$ that scales this vector:
  $$
  Av = \lambda v
  $$
  where the scalar $λ$ is called the **eigenvalue** corresponding to this eigenvector

  - **left eigenvector
    $$
    v^TA = \lambda v^T
    $$

  - **right eigenvector** [NTR_18)
    $$
    Av = \lambda v
    $$

- **eigendecomposition**: decomposition of a matrix into a set of eigenvectors and eigenvalues

  - The matrix $A$ has $n$ linearly independent eigenvectors $V = [v^{(1)}, ... , v^{(n)}]$ corresponding to $n$ eigenvalues $\lambda = [\lambda_1, ... , \lambda_n]$

  - The eigendecomposition of the matrix $A$
    $$
    A = Vdiag(\lambda)V^{-1}
    $$

  - In the case where $A$ is a real symmetric matrix, $V$ is an orthogonal matrix

    - $A$ is viewed as a space extending $\lambda_i$ times along the direction $v^{(i)}$



- **positive definite**: a matrix with all eigenvalues positive
  $$
  x^TAx = 0 \Rightarrow x = 0
  $$

- **positive semidefinite**: a matrix in which all eigenvalues are non-negative
  $$
  \forall x, \hspace{0.1cm} x^TAx \geq 0
  $$

- **negative definite**: a matrix in which all the eigenvalues are negative $$
  x^TAx = 0 \Rightarrow x = 0
  $$ **negative definite**: a matrix in which all the eigenvalues are negative, say

- **negative semidefinite**: a matrix in which all eigenvalues are nonpositive.



# 8. singular value decomposition

- **Singular Value Decomposition (SVD)**: decomposes a matrix into **singular vector** and **singular value**.

- Every real matrix has a singular value decomposition, but not always an eigendecomposition

  - Non-square matrices do not have an eigendecomposition, so we can only use the singular value decomposition.

- Singular value decomposition of matrix $A$
  $$
  A = UDV^T
  $$
  Where $A$ is a matrix of $m*n$, $U$ is a matrix of $m*m$, $D$ is a matrix of $m*n$, and $V$ is a matrix of $n*n$

  - $U$ and $V$ are both orthogonal matrices and $D$ is a diagonal matrix

  - The elements on the diagonal of $D$ are called the **singular values** of the matrix $A$.
  - The column vectors of $U$ are called the **left singular vectors**
    - Eigenvectors of [NTR_99
  - The column vector of $V$ is called **right singular vector**.
    - Eigenvectors of $A^TA$.
  - Non-zero singular value of $A$: square root of the eigenvalue of $AA^T$ or $A^TA$.



# 9. Moore-Penrose pseudoinverse

- **Moore-Penrose pseudoinverse**

- Definition of the pseudoinverse of matrix $A(m*n)$
  $$
  A^+ = \lim\limits_{\alpha \rightarrow 0} (A^TA + \alpha I)^{-1}A^T
  $$

- The actual algorithm for calculating the pseudoinverse is not based on this definition, but uses the following equation
  $$
  A^+ = VD^+U^T
  $$

  - where the matrices $U$, $D$ and $V$ are the matrices obtained by singular value decomposition of the matrix $A$
  - $D^+$: non-zero elements are obtained by taking the reciprocal and then transposing them

- When $m \leqslant n$, solving linear equations using pseudo-inverses is one of many possible solutions

  - $x=A^+y$ is the smallest of all feasible solutions to the equation $||x||_2$.

- When $m > n$, there may be no solution

  - $x$ obtained by pseudo-inverse makes $Ax$ and $y$ the smallest of $||Ax - y||_2$



# 10. trace operation

- Trace operation: sum of diagonal elements of a matrix
  $$
  Tr(A) = \sum_{i}A_{i,i}
  $$

- Another way to describe the Frobenius paradigm of a matrix
  $$
  ||A||_F = \sqrt{Tr(AA^T)} 
  $$

- The trace operation is invariant under the transpose operation
  $$
  Tr(A) = Tr(A^T)
  $$

- The trace of the square matrix obtained by multiplying several matrices is the same as the trace of the multiplication after moving the last of these matrices to the top
  $$
  Tr(ABC) = Tr(CAB) = Tr(BCA)
  $$



# 11. rows and columns

- Rowwise: a function that maps the square array $A$ to the real numbers
  - Denoted as $det(A)$.
  - Equals the product of the eigenvalues of the matrix
- The absolute value of the determinant can be used to measure how much the matrix expands or shrinks as a result of matrix multiplication.
- If the determinant is $0$, then the space has completely shrunk along at least one dimension, causing it to lose all of its volume
- If the determinant is $1$, then this transformation leaves the volume of space unchanged
