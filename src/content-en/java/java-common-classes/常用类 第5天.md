## 1. Use of BigDecimal

- double is an approximation of the value stored, does not meet the requirements of accurate calculations, you need to use BigDecimal
- Location: java.
- Role: precise calculation of floating-point numbers
- Creation method: BigDecimal bd = new BigDecimal(String str);; BigDecimal bd = new BigDecimal(String str)
- method:
  - BigDecimal add(BigDecimal bd): add
  - BigDecimal subtract(BigDecimal bd): minus
  - BigDecimal multiply(BigDecimal bd): multiplication
  - BigDecimal divide(BigDecimal bd, int scal, RoundingMode mode): divide
    - scal: Specify the exact number of decimal places.
    - mode:
      - Specify the decimal part of the rounding mode, usually using the rounding mode
      - ROUND\_HALF\_UP for BigDecimal.

## 2. Date class

- A Date represents a specific moment in time, down to the millisecond.
- Most of the methods in the Date class have been replaced by methods in the Calendar class
- Units of time
  - 1 second = 1000 milliseconds
  - 1 millisecond = 1000 microseconds
  - 1 microsecond = 1000 nanoseconds

```java
import java.util.Date;

public class Test {
    public static void main(String[] args) {
        // 创建Date对象
        // 今天
        Date date1 = new Date();
        System.out.println(date1);

        // 昨天
        Date date2 = new Date(date1.getTime() - 60 * 60 * 24 * 1000);
        System.out.println(date2);

        // 方法after, before, compareTo, equals
        System.out.println(date1.after(date2));
        System.out.println(date1.before(date2));
        System.out.println(date1.compareTo(date2));
        System.out.println(date1.equals(date2));
    }
}
```

## 3. Calendar Class

- Calendar provides methods to get or set various calendar fields
- Constructor
  - protected Calendar(): since the modifier is protected, it is not possible to create this object directly
- Other methods:
  - static Calendar getInstance(): get the calendar using the default time zone and region.
  - void set(int year, int month, int date, int hourofday, int minute,int second): set the calendar year, month, day, hour, minute and second
  - int get(int field): return the value of the given calendar field. Fields such as year, month, day, etc.
  - void setTime (Date date): set the time of this calendar with the given Date, Date-Calendar
  - Date getTime(): return a Date to indicate the time of this calendar, Calendar-Date
  - void add(int field, int amount): according to the rules of the calendar, to the specified field to add or reduce the amount of time
  - long getTimeInMillies (): milliseconds to return to the time value of the calendar

```java
import java.util.Calendar;

public class Test {
    public static void main(String[] args) {
        // 创建Calendar对象
        Calendar calendar = Calendar.getInstance();
        System.out.println(calendar.getTime());
        System.out.println(calendar.getTimeInMillis());

        // 获取时间信息
        // 获取年
        System.out.println(calendar.getWeekYear());
        System.out.println(calendar.get(Calendar.YEAR));

        // 获取月
        System.out.println(calendar.get(Calendar.MONTH) + 1);

        // 获取日
        System.out.println(calendar.get(Calendar.DATE));
        System.out.println(calendar.get(Calendar.DAY_OF_MONTH));

        // 获取小时
        System.out.println(calendar.get(Calendar.HOUR));
        System.out.println(calendar.get(Calendar.HOUR_OF_DAY));

        // 获取分钟
        System.out.println(calendar.get(Calendar.MINUTE));

        // 获取秒
        System.out.println(calendar.get(Calendar.SECOND));
    }
}
```

## 4. SimpleDateFormat class

- SimpleDateFormat is a concrete class that formats and parses dates in a language-dependent way
- Performs formatting (date -> text), parsing (text -> date)
- Common time pattern characters
  - y: year
  - M: month of the year
  - d: number of days in a month
  - H: number of hours in a day (0~23)
  - m: minutes
  - s: second
  - S: milliseconds

```java
import java.text.SimpleDateFormat;
import java.util.Date;

public class Test {
    public static void main(String[] args) throws Exception {
        // 创建SimpleDataFormat对象
        SimpleDateFormat sdf = new SimpleDateFormat("yyyy/MM/dd HH-mm-ss");

        // 创建date
        Date date = new Date();

        // 格式化date(时间 -> 字符串)
        System.out.println(sdf.format(date));

        // 解析(字符串 -> 时间)
        System.out.println(sdf.parse("2001/09/17 00-12-09"));
    }
}
```

## 5. System class

- The System class is mainly used to get the attribute data and other operations of the system.
- methods:
  - static void arraycopy(Object src, int srcPos, Object dest, int destPos, int length): copy array
    - src: the original array
    - srcPos: the subscript from which the original array is copied.
    - dest: target array
    - destPos: which subscript of the target array to overwrite.
    - length: length of the copy
  - static long current(TimeMillis()): get the current system time, the return value is milliseconds.
  - static void gc(): suggest that the JVM quickly start garbage collection period to collect garbage,
  - static void exit(int status): exit JVM, if the parameter is 0 means normal exit JVM, non-0 means abnormal exit JVM
