## 1. Classification of internal classes

- Internal Classes:
  - Member Internal Classes
  - Static Internal Classes
  - Local Internal Classes
  - Anonymous internal class

## 2. The concept of internal classes

- Concept: define another complete class inside a class (internal classes also generate class files)

  _```python
  class Outer{
      class Inner{
          
      }
  }
  ```
- Characteristics:

  - ```python
  class Outer{
      class Inner{
          
      }
  }
  ``` Features: ```python
  class Outer{
      class Inner{
          
      }
  }
  ``` compiles to a separate bytecode file.
  - Internal classes can directly access private members of external classes without breaking encapsulation.

    ```java
    public class Outer {
        private String name;

        class Inner {
            public void show() {
                System.out.println(name);
            }
        }
    }
    ```
  - Provides necessary internal functionality to the outer class

## 3. member inner classes

- A class defined inside a class, at the same level as instance variables and methods.
- An instance part of an external class that must be relied upon to create an object of the internal class.
- When an external class and an internal class have renamed attributes, the internal class attributes are accessed first.
- Member Internal classes cannot define static members, but they can define static constants.

```java
public class Test {
    public static void main(String[] args) {
        // 创建外部类对象
        Outer outer1 = new Outer();

        // 创建内部类对象
        Outer.Inner inner1 = outer1.new Inner();

        inner1.show();

        System.out.println("***************");

        // 简化
        Outer.Inner inner2 = new Outer().new Inner();

        inner2.show();
    }
}
```

```java
// 外部类
public class Outer {
    // 实例变量
    private String name = "ABC";
    private int age = 10;

    // 内部类
    class Inner {
        private String address = "qwe";
        private String phone = "123";
        private int age = 20;

        // 方法
        public void show() {
            // 打印外部类的属性
            System.out.println(Outer.this.name);
            System.out.println(Outer.this.age);

            // 打印内部类的属性
            System.out.println(this.address);
            System.out.println(this.phone);

            // 属性与外部类的属性同名，需要用 外部类名.this 调用
            System.out.println(this.age);
        }
    }
}
```

## 4. Static internal classes

- Does not depend on external class objects, can be created directly or accessed by class name, can declare static members
- Only internal classes can be modified with static

```java
public class Test {
    public static void main(String[] args) {
        // 直接创建静态内部对象
        Outer.Inner inner = new Outer.Inner();
        inner.show();
    }
}
```

```java
// 外部类
public class Outer {
    // 实例变量
    private String name = "ABC";
    private int age = 10;

    // 静态内部类（级别和外部类相同）
    static class Inner {
        private String address = "qwe";
        private String phone = "123";

        // 静态成员
        private static int num = 20;

        // 方法
        public void show() {
            // 调用外部类的属性
            // 先创建外部类对象
            Outer outer = new Outer();
            // 再调用外部类对象的属性
            System.out.println(outer.name);
            System.out.println(outer.age);

            // 调用静态内部类的属性和方法
            System.out.println(address);
            System.out.println(phone);
            System.out.println(num);
            System.out.println(Inner.num);
        }
    }
}
```

## 5. Local internal classes

- Defined in the method of the external class, the scope of action and object creation is limited to the current method.
- When a local internal class accesses a local variable in the current method of an external class, the variable must be qualified as final because there is no guarantee that the variable's lifetime will be the same as its own.
- Limit the scope of the class to the current method.

```java
public class Test {
    public static void main(String[] args) {
        Outer outer = new Outer();

        outer.show1();
    }
}
```

```java
// 外部类
public class Outer {
    private String name = "ABC";
    private int age = 10;

    public void show1() {
        // 定义局部变量
        String address = "qwe";

        // 局部内部类（不能加任何访问修饰符）
        class Inner {
            // 局部内部类的属性（不能有静态变量，但可以有静态常量）
            private String phone = "123";
            private final static int num = 20;

            public void show2() {
                // 访问外部类的属性
                System.out.println(Outer.this.name);
                System.out.println(Outer.this.age);

                // 访问内部类的属性
                System.out.println(this.phone);

                // 访问局部变量（jdk1.7要求：变量必须是常量，jdk1.8之后自动添加final）
                System.out.println(address);
            }
        }

        // 创建局部内部类对象
        Inner inner = new Inner();
        inner.show2();
    }
}
```

``` 6. Anonymous inner classes

- A local internal class without a class name (all characteristics are the same as a local internal class)
- Must inherit from a parent class or implement an interface
- The syntax for defining the class, implementing it, and creating objects is merged, and only one object of the class can be created.
- Advantages: reduces the amount of code
- Disadvantage: less readable

```java
public class Test {
    public static void main(String[] args) {
        // 创建接口类型的对象
        USB usb1 = new Mouse();
        usb1.service();

        // 局部内部类
        class Keyboard implements USB {

            @Override
            public void service() {
                System.out.println("Keyboard connect successfully");
            }
        }

        // 使用局部内部类创建对象
        USB usb2 = new Keyboard();
        usb2.service();

        // 匿名内部类进行优化（相当于创建了一个局部内部类）
        USB usb3 = new USB() {
            @Override
            public void service() {
                System.out.println("Connect successfully");
            }
        };
        usb3.service();
    }
}
```

```java
public class Mouse implements USB {
    @Override
    public void service() {
        System.out.println("Mouse connect successfully");
    }
}
```
