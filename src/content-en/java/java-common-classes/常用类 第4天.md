## 1. String Overview

- Strings are constants and cannot be changed after they are created
- String literals are stored in the string pool and can be shared
- String string = "ABC"; generate one object, stored in string pool
- String string = new String("ABC"); produces two objects, one in the heap and one in the pool

```java
public class Test {
    public static void main(String[] args) {
        // 常量存储在字符串池中
        String string1 = "ABC";

        // 给字符串赋值时，没有修改数据，而是在字符串池中重新开辟空间
        string1 = "qwe";

        String string2 = new String("ABC");
    }
}
```

## 2. String common methods

- public int length(): return the length of the string
- public char charAt(int index): according to the subscript to get the character
- public boolean contains (String string): determine whether the current string contains string

```java
public class Test {
    public static void main(String[] args) {
        String string1 = "ABCD";
        System.out.println(string1.length());
        System.out.println(string1.charAt(2));
        System.out.println(string1.contains("BC"));
    }
}
```

- public char[] toCharArray(): convert the string into an array
- public int indexOf(String string): find the first occurrence of the string subscripts, the existence of the return of the subscripts, does not exist then return -1
- public int lastIndexOf (String string): find the last occurrence of the string in the current string subscript index

```java
public class Test {
    public static void main(String[] args) {
        String string1 = "ABCBCD";
        System.out.println(string1.toCharArray());
        System.out.println(string1.indexOf("BC"));
        System.out.println(string1.indexOf("BC", 2));
        System.out.println(string1.lastIndexOf("BC"));
    }
}
```

- public String trim (): remove spaces before and after the string
- public String toUpperCase (): the lower case to upper case
- public boolean endWith(String string): determine whether the string ends with string

```java
public class Test {
    public static void main(String[] args) {
        String string1 = " AbCD";
        System.out.println(string1.trim());
        System.out.println(string1.toUpperCase());
        System.out.println(string1.endWith("CD"));
    }
}
```

- public String replace(char OldChar, char NewChar): replace the old character/string with a new character/string
- public String[] split(String string): according to the string to do splitting
- public boolean equals(String string): compare the given string with the original string is the same or not
- public int compareTo(String string): Calculate the given string and the original string in accordance with the order of the first different characters between the difference between the value, if exactly the same but there are more characters, the length of the string will calculate the difference between the values

```java
public class Test {
    public static void main(String[] args) {
        String string1 = "ABCDE";
        String string2 = "ABC";

        for (Object i : string1.split("A")) {
            System.out.println(i);
        }

        System.out.println("*********");

        for (Object i : string1.split("[AB]")) {
            System.out.println(i);
        }

        System.out.println("*********");

        for (Object i : string1.split("[AB]+")) {
            System.out.println(i);
        }

        System.out.println(string1.equals(string2));
        System.out.println(string1.equalsIgnoreCase(string2));

        System.out.println(string1.compareTo(string2));

    }
}
```

## 3. StringBuffer and StringBuilder

- StringBuffer: variable-length strings, JDK1.0 provides, slow running efficiency, thread-safe
  - append(): appending
  - insert (): add
  - replace(): replace
  - delete(): delete

```java
public class Test {
    public static void main(String[] args) {
        StringBuffer stringBuffer = new StringBuffer();

        // append()：追加
        stringBuffer.append("ABC");
        stringBuffer.append("abc");

        // insert()：添加
        stringBuffer.insert(0, "123");

        // replace()：替换
        stringBuffer.replace(0, 3, "246");

        // delete()：删除
        stringBuffer.delete(0, 3);

        System.out.println(stringBuffer);

    }
}
```

- StringBuilder: variable-length strings, JDK5.0 provides, running fast, thread unsafe
  - append(): appending
  - insert(): add
  - replace(): replace
  - delete(): delete

```java
public class Test {
    public static void main(String[] args) {
        StringBuilder stringBuilder = new StringBuilder();

        // append()：追加
        stringBuilder.append("ABC");
        stringBuilder.append("abc");

        // insert()：添加
        stringBuilder.insert(0, "123");

        // replace()：替换
        stringBuilder.replace(0, 3, "246");

        // delete()：删除
        stringBuilder.delete(0, 3);

        System.out.println(stringBuilder);

    }
}
```

- Difference with String:
  - More efficient than String
  - Saves memory compared to String

```java
public class Test {
    public static void main(String[] args) {
        long start1 = System.currentTimeMillis();
        String string1 = "";
        for (int i = 0; i < 5000; i++) {
            string1 += i;
        }
        System.out.println(string1);
        long end1 = System.currentTimeMillis();
        System.out.println(end1 - start1);

        long start2 = System.currentTimeMillis();
        StringBuffer string2 = new StringBuffer();
        for (int i = 0; i < 5000; i++) {
            string2.append(i);
        }
        System.out.println(string2);
        long end2 = System.currentTimeMillis();
        System.out.println(end2 - start2);

        long start3 = System.currentTimeMillis();
        StringBuilder string3 = new StringBuilder();
        for (int i = 0; i < 5000; i++) {
            string3.append(i);
        }
        System.out.println(string3);
        long end3 = System.currentTimeMillis();
        System.out.println(end3 - start3);
    }
}
```
