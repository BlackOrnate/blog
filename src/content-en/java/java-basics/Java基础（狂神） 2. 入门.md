## 1. The Birth of the Java Empire

- C & C++
  - C was born in 1972.
    - Close to hardware, very fast, very efficient
    - Operating systems, compilers, databases, network systems, etc.
    - Pointers and memory management
  - 1982 C++ was born
    - Object-oriented
    - Compatible with C
    - Graphics, games, etc.
- Java is born
  - 1995 Web pages were simple and crude, lacking interactivity
  - Programs with graphical interfaces (Applets)
  - Java 2 Standard Edition: desktop
  - Java 2 Mobile: cell phones
  - Java 2 Enterprise Edition: server
- Java Development
  - Many platforms, systems, and tools have been developed based on Java.
    - Build tools: Ant, Maven, Jekins
    - Application Servers: Tmocat, Jetty, Jboss, Websphere, weblogic
    - Web Development: Struts, Spring, Hibernate, myBatels
    - Development Tools: Eclipse, Netbean, intellij idea, Jbuilder
  - 2006: Hadoop (big data field)
  - 2008: Android (cell phone)

## 2. Java features and benefits

- Simplicity
- Object Oriented
- Portability
- High Performance
- Distributed
- Dynamic
- Multi-threaded
- Security
- Robustness

## 3. The three major versions of Java

- Write Once, Run Anywhere
- JavaSE: Standard Edition (desktop programs, console development)
- JavaME: embedded development (cell phones, small appliances)
- JavaEE: E enterprise-level development (web, server development)

- JDK: Java Development Kit
- JRE: Java Runtime Environment
