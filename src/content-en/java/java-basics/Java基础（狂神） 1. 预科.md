## 1. What is a computer

- Computer: full name electronic computer, commonly known as computer
- Able to run in accordance with the **program**, automatic, high-speed processing of large amounts of data can only be modernized electronic equipment.
- By the **hardware** and **software** composed of
- Common forms of desktop, laptop computers, mainframe computers, etc.
- Widely used in: scientific computing, data processing, automatic control, computer-aided design, artificial intelligence, networks and other fields.

## 5. Basic Dos Commands

- Disk switching: d:
- View all files in the current directory: dir
- Switch directories:
  - Switch across disks: cd /d Folder name
  - Return to previous directory: cd ...
  - Go to folder: cd folder name (first few characters of folder + TAB)
- Clear screen: cls
- Exit: exit
- Check your computer's IP: ipconfig
- Open computer: calc
- Open drawing tablet: mspaint
- Open notepad: notepad
- Get IP information about a website: ping www.bilibili.com
- Create a folder: md Name of folder
- Delete a folder: rd folder
- Create a file: cd> full name of the file
- Delete a file: del File name

## 6. History of computer language development

- First Generation Languages

  - Machine languages
    - The basic calculations of a computer are based on binary
    - It is fed directly into the computer without any conversion.
- Second Generation Languages

  - Assembly language
    - Solves the problem of humans not being able to read machine language.
    - Instructions instead of binary
  - Current application:
    - Reverse engineering
    - Robotics.
    - Viruses.
- Third-generation languages

  - Moore's Law
  - At a constant price, the number of transistors that can fit on an integrated circuit doubles every 18 months, doubling its performance.
  - High-level languages
  - High-level languages are broadly divided into two categories: **procedure-oriented** and **object-oriented**.
  - C is a typical process-oriented language
  - C++ and JAVA are typical object-oriented languages.
  - Various languages:

    - C language
    - C++ language
    - JAVA language
    - C# language
    - Python, PHP, JavaScript
