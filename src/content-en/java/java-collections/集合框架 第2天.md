## 1. An Overview of Panotypes

- Java generic is a new feature introduced in JDK1.5, the essence of which is to parameterize the type, passing the type as a parameter.
- Common forms are generic classes, generic interfaces, generic methods
- Syntax:
  - <T,...> : T is called a type placeholder, representing a reference type
- Benefits:
  - Improve code reusability
  - Prevent type conversion exceptions and improve code security.

## 2. generic classes

- Caution:
  - A generic can only be a reference type.
  - Different generic objects cannot be assigned to each other

```java
public class Test {
    public static void main(String[] args) {
        // 使用泛型类创建对象
        TestGenericClass<String> testGeneric1 = new TestGenericClass<String>();
        testGeneric1.t = "ABC";
        testGeneric1.show("abc");
        System.out.println(testGeneric1.getT());

        TestGenericClass<Integer> testGeneric2 = new TestGenericClass<>();
        testGeneric2.t = 10;
        testGeneric2.show(100);
        System.out.println(testGeneric2.getT());
    }
}
```

```java
public class TestGenericClass<T> {
    // 使用泛型
    // 创建变量
    T t;

    // 作为方法参数
    public void show(T t) {
        System.out.println(t);
    }

    // 作为方法返回值
    public T getT() {
        return t;
    }
}
```

## 3. Generic interfaces

- Caution:
  - Static constants cannot be created using generics

```java
public class Test {
    public static void main(String[] args) {
        TestInterface1 testInterface1 = new TestInterface1();
        System.out.println(testInterface1.server("abc"));

        TestGenericInterface<String> testGeneric = new TestInterface1();
        System.out.println(testGeneric.server("qwe"));

        TestInterface2<Integer> testInterface2 = new TestInterface2<>();
        System.out.println(testInterface2.server(999));
    }
}
```

```java
public class TestInterface1 implements TestGenericInterface<String> {
    @Override
    public String server(String s) {
        System.out.println(s);
        return s + "123";
    }
}
```

```java
public class TestInterface2<T> implements TestGenericInterface<T> {
    @Override
    public T server(T t) {
        System.out.println(t);
        return t;
    }
}
```

```java
public interface TestGenericInterface<T> {
    String name = "ABC";

    T server(T t);
}
```

## 4. Generic methods

```java
public class Test {
    public static void main(String[] args) {
        TestGenericMethod testGenericMethod = new TestGenericMethod();
        testGenericMethod.show("ABC");
        testGenericMethod.show(123);
    }
}
```

```java
public class TestGenericMethod {
    // 泛型方法
    public <T> T show(T t) {
        System.out.println("ABC" + t);
        return t;
    }
}
```

## 5. Generalized collections

- Concept: parameterized type, type-safe collection, forces collection elements to be of the same type
- Features:
  - Can be checked at compile time instead of throwing exceptions at runtime.
  - Access without type conversion (unboxing).
  - References cannot be assigned to each other between different generics, there is no polymorphism in generics

```java
import java.util.ArrayList;
import java.util.Iterator;

public class Test {
    public static void main(String[] args) {
        ArrayList<String> arrayList1 = new ArrayList<>();
        arrayList1.add("ABC");
        arrayList1.add("abc");
        for (String str : arrayList1) {
            System.out.println(str);
        }

        ArrayList<Student> arrayList2 = new ArrayList<>();
        Student student1 = new Student("A", 10);
        Student student2 = new Student("B", 20);
        Student student3 = new Student("C", 30);
        arrayList2.add(student1);
        arrayList2.add(student2);
        arrayList2.add(student3);

        Iterator<Student> iterator = arrayList2.iterator();
        while (iterator.hasNext()) {
            System.out.println(iterator.next());
        }

    }
}

class Student {
    String name;
    int age;

    Student(String name, int age) {
        this.name = name;
        this.age = age;
    }

    @Override
    public String toString() {
        return "Student{" +
                "name='" + name + '\'' +
                ", age=" + age +
                '}';
    }
}
```
