## 1. Throwing exceptions

- raise: throwing an exception
- Can only be thrown in a try block.
- You can't have a function throw an exception like you can in java.

```python
def calculate(i, j):
    if j == 0:
        raise Exception("Cannot calculate")

    print(i / j)

try:
    calculate(5, 0)
except Exception as e:
    print(str(e))
else:
    print("Finish")
```

## 2. Get the traceback string

- traceback.format\_exc(): get traceback information

```python
import traceback

try:
    raise Exception("Exception")
except Exception as e:
    print(traceback.format_exc())
```

## 4. logs

### 4.1 Using the logging module

- logging.basicConfig(): set the log format.
- logging.debug(): call logging.basicConfig() to output information.

```python
import logging

logging.basicConfig(
    level=logging.DEBUG,
    format='%(asctime)s - %(levelname)s - %(message)s')

logging.debug('Starting')

def test(n):
    logging.debug(f'Starting logging({n})')
    total = 1
    for i in range(1, n + 1):
        total *= i
        logging.debug(f"i = {i},total = {total}")
    logging.debug(f'Finished logging({n})')
    return total

print(test(5))
logging.debug('Finished')
```: call logging.basicConfig() to output information.

### 4.2 Logging Levels

| Level | Logging Function | Description |
| --- | --- | --- |
| DEBUG | logging.debug() | minor details |
| INFO | logging.info() | Logging information about general events in the program |
| WARNING | logging.warning() | Indicates possible problems |
| ERROR | logging.error() | logging errors |
| CRITICAL | logging.critical() | indicates fatal errors |

### 4.3 Disabling Logging

- logging.disable(): disables all messages after this line of code

```python
import logging

logging.basicConfig(
    level=logging.INFO,
    format='%(asctime)s - %(levelname)s - %(message)s')

logging.critical("Critical")

logging.disable(logging.CRITICAL)

logging.critical("Critical")
logging.error("Error")
```

### 4.4 Logging to a file

```python
import logging

logging.basicConfig(
    filename="./logging.txt",
    level=logging.INFO,
    format='%(asctime)s - %(levelname)s - %(message)s')
logging.critical("Critical")
logging.critical("Critical")
logging.error("Error")
```
