## 1. Finding text patterns without regular expressions

```python
def isPhoneNumber(text):
    if len(text) != 11:
        return False

    for i in range(0, len(text)):
        if (i == 3 or i == 7) and text[i] != "-":
            return False
        elif i != 3 and i != 7 and not text[i].isdecimal():
            return False

    return True

text = "123-456-789"
print(text)
print(isPhoneNumber(text))
```

## 2. Use regular expressions to find text patterns.

### 2.1 Creating Regular Expression (Regex) Objects

```python
import re

text = re.compile(r'\d\d\d-\d\d\d-\d\d\d')
```

### 2.2 Matching Regex Objects

```python
import re

text = re.compile(r'\d\d\d-\d\d\d-\d\d\d')
match = text.search("~~~123-456-789~~~")
print(match.group())
```

## 3. Matching more patterns with regular expressions

### 3.1 Grouping with Parentheses

```python
import re

text = re.compile(r'(\d\d\d)-(\d\d\d-\d\d\d)')
match = text.search("~~~123-456-789~~~")
print(match.group(1))
# 123
print(match.group(2))
# 456-789
print(match.groups())
# ('123', '456-789')
```

### 3.2 Matching multiple groupings with pipes

- | : Pipeline

```python
import re

text = re.compile(r'456|123')
match = text.search("123-456-789")
print(match.group())
# 123
```

### 3.3 Optional matching with question marks

```python
import re

text = re.compile(r'\d\d\d(~)?\d\d\d')
match = text.search("123123")
print(match.group())
# 123123

match = text.search("123~123")
print(match.group())
# 123~123
```

### 3.4 Matching zero or more times with an asterisk

```python
import re

text = re.compile(r'\d\d\d(~)*\d\d\d')
match = text.search("123123")
print(match.group())
# 123123

match = text.search("123~~~123")
print(match.group())
# 123~~~123
```

### 3.5 Match one or more times with plus sign

```python
import re

text = re.compile(r'\d\d\d(~)+\d\d\d')
match = text.search("123~123")
print(match.group())
# 123~123

match = text.search("123~~~123")
print(match.group())
# 123~~~123
```

### 3.6 Matching a specific number of times with braces

```python
import re

text = re.compile(r'\d\d\d(~){3,5}\d\d\d')
match = text.search("123~~~123")
print(match.group())
# 123~~~123

match = text.search("123~~~~~123")
print(match.group())
# 123~~~~~123
```

## 4. Greedy and non-greedy matching

- Greedy matching: matches the longest string possible.
- Non-greedy matching: matches the shortest string possible.

```python
import re

text = re.compile(r'(123 ){2,4}')
match = text.search("123 123 123 123 123 ")
print(match.group())
# 123 123 123 123

text = re.compile(r'(123 ){2,4}?')
match = text.search("123 123 123 123 123 ")
print(match.group())
# 123 123
```

## 5. findall() method

```python
import re

text = re.compile(r'\d\d\d-\d\d\d-\d\d\d')
match = text.search("~~~123-456-789~~~111-222-333~~~")
print(match.group())
# 123-456-789

match = text.findall("~~~123-456-789~~~111-222-333~~~")
print(match)
# ['123-456-789', '111-222-333']
```

## 6. Character sorting

| Write character classification | Representation |
| --- | --- |
| \D | Any number from 0 to 9 |
| \D | Any character other than a number from 0 to 9 |
| \w | Any letters, numbers and underscore characters |
| \W | Any character except letters, numbers and underscore characters |
| \s | A space, tab or line break |
| \S | Any character except spaces, tabs and line breaks |

## 7. Create your own character categories

```python
import re

text = re.compile(r'[0-5]')
match = text.findall("1a2b3c4d")
print(match)
# ['1', '2', '3', '4']

text = re.compile(r'[abc]')
match = text.findall("1a2b3c4d")
print(match)
# ['a', 'b', 'c']

text = re.compile(r'[^abc]')
match = text.findall("1a2b3c4d")
print(match)
# ['1', '2', '3', '4', 'd']
```

## 8. Insert characters and dollar signs

- ^ : starts with the specified text
- $ : ends with the specified text

```python
import re

text = re.compile(r'^\d\d\d')
match = text.search("123abc456")
print(match)
# <re.Match object; span=(0, 3), match='123'>

text = re.compile(r'\d\d\d$')
match = text.search("123abc456")
print(match)
# <re.Match object; span=(6, 9), match='456'>
```

## 9. Wildcard characters

- . : Matches all characters except newlines

```python
import re

text = re.compile(r'..23')
match = text.findall("123abc23")
print(match)
# ['bc23']
```

### 9.1 Match all characters with dot-star

```python
import re

text = re.compile(r'123(.*)456(.*)')
match = text.findall("123abc456def")
print(match)
# [('abc', 'def')]
```

### 9.2 Matching newlines with period characters

- re.DOTALL: Match all characters (including newlines) with dot characters.

```python
import re

text = re.compile(r'.*')
match = text.search("123abc\n456def")
print(match.group())
# 123abc

text = re.compile(r'.*', re.DOTALL)
match = text.search("123abc\n456def")
print(match.group())
# 123abc\n456def
```

## 10. case-insensitive matching

- re.I : case-insensitive

```python
import re

text = re.compile(r'abc', re.I)
match = text.findall("abcABC")
print(match)
# ['abc', 'ABC']
```

## 11. Replacing strings with the sub() method

```python
import re

text = re.compile(r'ABC\w*')
match = text.sub("abc", "ABC : 123")
print(match)
# abc : 123
```
