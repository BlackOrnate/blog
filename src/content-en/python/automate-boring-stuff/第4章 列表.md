## 1. List Data Types

- Lists: equivalent to arrays

```python
ABC = ["A", "B", "C"]

print(ABC)
```

### 1.1 Getting a single value in a list by indexing it

```python
ABC = ["A", "B", "C"]

print(ABC[0])
```

### 1.2 Negative indexing

- Negative index: lookup from back to front

```python
ABC = ["A", "B", "C"]

print(ABC[-1])
```

### 1.3 Getting sublists with slicing

- First parameter: start index, if not written, it starts from the beginning of the list.
- The second parameter: the ending index, without it, slicing to the end of the list
- Slices the list from the start index to the end index (without the end index) and forms a new list.

```python
values = [1, 2, 3, 4, 5, 6, 7]

print(values[1:5])
```

### 1.4 Getting the length of a list with the len() function

```python
ABC = ["A", "B", "C"]

print(len(ABC))
```

### 1.5 Changing values in a list by index

```python
ABC = ["A", "B", "C"]

ABC[1] = "b"
print(ABC)
```

### 1.6 Joining lists

```python
ABC = ["A", "B", "C"]

print(ABC + ["A", "B", "C"])
```

### 1.7 List copying

```python
values = [1, 2, 3, 4, 5, 6, 7]

a = values[:]
values[0] = 123
print(values)
print(a)
# 不能将value直接赋值给a
# 这样操作仅仅是指针传递
```

### 1.8 Removing values from a list with the del statement

```python
ABC = ["A", "B", "C"]

del ABC[0]
print(ABC)
```

## 2. Using Lists

### 2.1 Lists for loops

```python
ABC = ["A", "B", "C"]

for i in ABC:
    print(i)
```

### 2.2 In and not in operators

```python
ABC = ["A", "B", "C"]

print("A" in ABC)
print("A" not in ABC)
```

### 2.3 Using the enumerate() function with lists

- enumerate() function: returns the index of the element and the element itself

```python
ABC = ["A", "B", "C"]

for index, item in enumerate(ABC):
    print(f"{index} -> {item}")
    
print(list(enumerate(ABC)))
```

### 2.4 random.choice() and random.shuffle() functions with lists

```python
import random

ABC = ["A", "B", "C"]

print(random.choice(ABC))

print(ABC)
random.shuffle(ABC)
print(ABC)
```

## 4. Methods

### 4.1 Finding values in a list with the index() method

```python
ABC = ["A", "B", "C", "B"]

print(ABC.index("B"))
```

### 4.2 Adding values to a list with the append() method

```python
ABC = ["A", "B", "C"]

ABC.append("D")
print(ABC)
```

### 4.3 Adding values to a list with the insert() method

- Prefixes the selected subscript with

```python
ABC = ["A", "B", "C"]

ABC.insert(0, "D")
print(ABC)
```

### 4.4 Removing values from a list with the remove() method

- Only the first specified value is removed

```python
ABC = ["B", "A", "B", "C"]

ABC.remove("B")
print(ABC)
```

### 4.5 Removing values from a list with the pop() method

- Removes the element with the specified subscript
- Defaults to the end of the list if no subscript is specified.

```python
ABC = ["A", "B", "C"]
ABC.pop(1)
print(ABC)
```

### 4.6 Sorting values in a list with the sort() method

- A list consisting of letters only is sorted from smallest to largest by the ANSCII code of the first letter.
- A list consisting of only numbers is sorted from smallest to largest by the ANSCII code of the first letter.
- Mixed lists cannot be sorted.

```python
ABC = ["B", "A", "B", "C"]
ABC.sort()
print(ABC)
```

- Can be changed to reverse by passing reverse=True.

```python
ABC = ["B", "A", "B", "C"]
ABC.sort(reverse=True)
print(ABC)
```

### 4.7 Temporarily sorting values in a list with the sorted() method

```python
ABC = ["B", "A", "B", "C"]

print(ABC.sorted())
print(ABC)
```

### 4.7 Reversing values in a list with the reverse() method

```python
ABC = ["A", "B", "C"]

print(ABC.reverse())
```

## 5. Sequence Data Types

## 5.1 Variable and immutable data types

- List: Variable data type
- String: immutable data type

### 5.2 Tuple data types

- A tuple is similar to a list, but a tuple cannot be modified.
- Tuples are similar to lists, but tuples cannot be modified, neither the number of elements, nor the values of the elements can be modified.

```python
values = (1, 2, 3, 4, 5, 6, 7)

print(values[0])
```

### 5.3 Converting Types with the list() and tuple() Functions

```python
print(tuple([1, 2, 3]))
print(list((1, 2, 3)))
print(list("123"))
```

## 6. References

### 6.1 Identification and the id() function

- id() function: get address

```python
str = "ABC"

print(id(str))
```

### 6.2 Passing References

```python
def ABC(list):
    list.append(123)

list = [1, 2, 3]
print(list)
ABC(list)
print(list)
```

### 6.3 The copy() and deepcopy() functions of the copy module

- If the list to be copied is a nested list, use deepcopy().

```python
import copy

ABC = ["A", "B", "C"]

abc = copy.copy(ABC)
ABC[0] = "a"
print(ABC)
print(abc)
```
