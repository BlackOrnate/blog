## 1. def statement and arguments

```python
def ABC(message):
    print(message)

ABC(123)
```

- Formal parameter: message
- Real parameter: 123

## 2. Return values and return statements

```python
def ABC(message):
    return message

print(ABC(123))
```

## 3. None value

- None value: null

```python
print(print("ABC") is None)
```

## 4. Keyword arguments and the print() function

- end: end with the specified string, default is "\n".

```python
print("A", end="")
print("B")
```

- sep: each string is concatenated with the specified string, default is ""

```python
print("A", "B", "C", sep=", ")
```

## 5. Local and global scopes

## 5.1 Local variables cannot be used in global scopes

```python
def ABC():
    a = 1
    
    
ABC()
print(a)
```

### 5.2 Local scopes cannot use variables in other local scopes

```python
def ABC():
    a = 1
    abc()
    print(a)

def abc():
    a = 2

ABC()
```

### 5.3 Global variables can be read in local scopes

```python
def ABC():
    print(a)

a = 1
ABC()
```

### 5.4 Local and global variables with the same name

```python
def ABC():
    a = 1
    print(a)
    abc()
    print(a)

def abc():
    a = 2
    print(a)

a = 0
ABC()
print(a)
```

## 6. global statement

- global: Declares the variable to be a global variable.

```python
def ABC():
    global a
    a = 1

def abc():
    a = 2
    print(a)
    ABC()
    print(a)

abc()
print(a)
```

## 7. Exception handling

- try-except-else: equivalent to try-catch-else

```python
try:
    print(5/0)
except ZeroDivisionError:
    print("Error")
else:
    print("Finish")
```

- try: the block of code to try
- except: the block of code that is executed when the corresponding exception occurs.
  - You can set the block to pass to skip the exception handling.
- else: the code block to be executed when there is no such exception.
