## 4. Control Flow Statements

### 4.1 if statements

```python
a = 1
if a == 1:
    print("a = 1")
```

### 4.2 else statement

```python
a = 1
if a == 1:
    print("a = 1")
else:
    print("False")
```

### 4.3 elif statement

```python
a = 2
if a == 1:
    print("a = 1")
elif a == 2:
    print("a = 2")
else:
    print("False")
```

### 4.4 while loops

```python
a = 1
while a < 5:
    a += 1
    print(a)
```

### 4.5 break statement

```python
a = 1
while a < 5:
    a += 1
    if a == 3:
        break
    print(a)
```

### 4.6 continue statement

```python
a = 1
while a < 5:
    a += 1
    if a == 3:
        continue
    print(a)
```

### 4.7 For loops and the range() function

- for loops: Enhancing for loops
- The range() function
  - First argument: the lower bound of the value of the list element
  - Second argument: the upper limit of the value of the list element
  - third parameter: step size, default is 1

```python
for value in range(1, 10, 2):
    print(value)
```

## 5. Importing modules

### 5.1 The import statement

- test.py

```python
def ABC(message):
    print(message)
```

- main.py

```python
import test

```

### 5.2 from import statement

#### 5.2.1 Importing a Specified Function

- test.py

```python
def ABC(message):
    print(message)
```

- main.py

```python
from test import ABC

ABC(1)
```

#### 5.2.2 Importing all functions

- test.py

```python
def ABC(message):
    print(message)
```

- main.py

```python
from test import *

ABC(1)
```

## 6. Use the sys.exit() function to end the program early

```python
import sys

while True:
    message = input("Input 0 to exit: ")
    if message == "0":
        sys.exit()
    print(message)
```
