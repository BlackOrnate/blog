## 1. Handling strings

### 1.1 String literals

#### 1.1.1 Double quotes

```
str = "ABC"
```

#### 1.1.2 Escape characters

| Escape Characters |
| --- |
| ' |
| " |
| \n |
| \t |
| \bord0\shad0\alphaH3D \bord0\shad0\alphaH3D

#### 1.1.3 Raw String

- Raw strings: ignore escaped characters

```python
str = r"ABC\n"
print(str)
```

#### 1.1.4 Multi-line strings with triple quotes

- Strings are written in exactly the same format as

```python
str = ''' abc
 a 1 \n
 b 2 \n
 c 3 \n
 123'''

print(str)
```

### 1.2 String indexing

```python
message = "ABC"
print(message[1])
```

### 1.3 String slicing

```python
message = "ABC"
print(message[0:2])
```

### 1.4 In and not in operators on strings

```python
message = "ABC"
print("A" in message)
print("A" not in message)
```

## 2. Putting strings into other strings

```python
A = "A"
B = "B"

print(A + ": " + B)
print("%s: %s" % (A, B))
print("{}: {}".format(A, B))
print(f"{A}: {B}")
```

## 3. Useful String Methods

### 3.1 The upper() method

```python
message = "abc"
print(message.upper())
```

### 3.2 lower() method

```python
message = "ABC"
print(message.lower())
```

### 3.3 isupper() method

```python
message = "aBC"
print(message.isupper())
```

### 3.4 islower() method

```python
message = "aBc"
print(message.islower())
```

### 3.5 isX() String method

| String Methods | Explanations |
| --- | --- |
| isalpha() | Whether to include only letters |
| isalnum() | whether to include only letters and numbers |
| isdecimal() | Is it numeric only || isspace()
| isspace() | if only whitespace (tabs, newlines, spaces) || istitle()
| istitle() | if it starts with a capital letter | | isspace()

### 3.6 startwith() and endwith() String Methods

```python
message = "ABC"
print(message.startswith("AB"))
print(message.endswith("BC"))
```

### 3.7 The join() string method

```python
message = "ABC"
print(message.join(["1", "2", "3"]))
```

### 3.8 split() String method to split a string

- Defaults to whitespace (tabs, newlines, spaces).
- Defaults to all separators.

```python
message = "ABCABC"
print(message.split("B", 1))
```

### 3.9 Using the partition() method to separate strings

```python
message = "ABCABC"
print(message.partition("B"))
```

### 3.10 Aligning text with the rjust(), ljust(), and center() methods

```python
message = "ABC"
print(message.rjust(10, "~"))
# ~~~~~~~ABC
print(message.ljust(10, "~"))
# ABC~~~~~~~
print(message.center(10, "~"))
# ~~~ABC~~~~
```

### 3.11 Remove whitespace with strip(), rstrip(), and lstrip().

```python
message = "  ABC\n"
print(message.rstrip())
#   ABC
print(message.lstrip())
# ABC
#
print(message.strip())
# ABC
```
