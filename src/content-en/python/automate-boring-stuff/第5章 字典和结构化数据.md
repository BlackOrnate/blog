## 1. Dictionary data types

- Key-value pairs
- Unordered

```
AB = {"A": 1, "B": 2}
```

### 1.1 Adding key-value pairs

```python
AB = {"A": 1, "B": 2}
AB["C"] = 3
print(AB)
```

### 1.2 Deleting key-value pairs

```python
AB = {"A": 1, "B": 2}
del AB["A"]
print(AB)
```

### 1.3 The keys() method

- Get the list of dictionary keys

```python
AB = {"A": 1, "B": 2}
for key in AB.keys():
    print(key)
```

### 1.4 values() method

```python
AB = {"A": 1, "B": 2}
for value in set(AB.values()):
    print(value)
```

### 1.5 items() method

- Get a list of key-value pairs for the dictionary

```python
AB = {"A": 1, "B": 2}
for key, value in AB.items():
    print("%s: %s" % (key, value))
```

### 1.6 Checking if a key or value exists in the dictionary total

```python
AB = {"A": 1, "B": 2}
print("A" in AB.keys())
print(1 in AB.values())
```

### 1.7 The get() method

```python
AB = {"A": 1, "B": 2}
print(AB.get("C", "Not Find"))
```

### 1.8 setdefault() method

- Checks to see if the key exists, and adds the key-value pair if it doesn't.

```python
AB = {"A": 1, "B": 2}
AB.setdefault("C", 3)
print(AB)
AB.setdefault("C", 4)
print(AB)
```

## 2. Nesting

### 2.1 Storing Dictionaries in Lists

```python
AB = {"A": 1, "B": 2}
ab = {"a": 1, "b": 2}
ABab = [AB, ab]
print(ABab)
```

### 2.2 Storing a list in a dictionary

```python
AB = {"A": 1, "B": ["a", "b"]}
print(AB)
```

### 2.3 Storing a dictionary in a dictionary

```python
AB = {"A": {"a": 1}, "B": {"b": 2}}
print(AB)
```
