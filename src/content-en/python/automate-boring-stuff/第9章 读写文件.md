## 1. Files and file paths

### 1.1 Generating file paths

```python
from pathlib import Path

myFiles = ["C++", "Java", "Python"]

for filename in myFiles:
    print(Path("D:\\Study", filename))
```

### 1.2 Current working directory (.cwd())

```python
from pathlib import Path

print(Path.cwd())
```

### 1.3 Home directory (.home())

```python
from pathlib import Path

print(Path.home())
```

### 1.4 Creating new folders with os.makedirs()

- Creates all folders in the directory

```python
import os

os.makedirs(".\\ABC\\abc")
```

- Only one folder can be created

```python
from pathlib import Path

Path(".\\ABC").mkdir()
```

### 1.5 Handling absolute and relative paths

- is\_absolute(): determine if the path is absolute or not

```python
from pathlib import Path

file = Path.cwd()
print(file)
print(file.is_absolute())
```

- os.path module
  - abspath(): get absolute path
  - isabs(): determine if it is an absolute path.
  - relpath(): get the relative path from the back file to the front file, the back file is the current working directory by default.

```python
import os.path

file = "..\\"
print(os.path.abspath(file))
print(os.path.isabs(file))
print(os.path.relpath("D:\\Study\\Java", "D:\\Study\\C++"))
```: get the relative path from the back file to the front file, back file is the current working directory by default.

### 1.6 Getting the parts of a file path

- Anchor: The root folder of the file system.
- drive: physical hard disk drive or other storage device
- parent: The folder containing the file.
- Name: the name of the file
  - Trunk name (stem)
  - Suffix

```python
from pathlib import Path

file = Path(Path.cwd(), "test.py")
print(file.anchor)
print(file.drive)
print(file.parent)
print(file.name)
print(file.stem)
print(file.suffix)
```.

### 1.7 Viewing file sizes and folder contents

```python
import os.path
from pathlib import Path

print(os.path.getsize(Path(Path.cwd(), "test.py")))
print(os.listdir(Path.cwd()))
```

### 1.8 Modifying a file list using wildcard patterns

```python
from pathlib import Path

file = Path.cwd()
print(list(file.glob("*.py")))
```

### 1.9 Checking the validity of paths

- exists(): determines if the path exists
- is\_file(): determines if the path exists and if it is a file.
- is\_dir(): determines if the path exists and if it is a folder.

```python
from pathlib import Path

file = Path.cwd()
print(file.exists())
print(file.is_file())
print(file.is_dir())
```

## 2. File reading and writing process

### 2.1 Opening a file with the open() function

```
open("messages.txt")
```

### 2.2 Reading the contents of a file

- read(): outputs the entire contents
- readlines(): saves the contents of the file in an array by line (newline characters are preserved).

```python
message = open("messages.txt")
print(message.read())
print(message.readlines())
message.close()
```

### 2.3 Writing to a file

- Mode setting of open()
  - "r": read-only (default)
  - "w": write-only
  - "a": append
  - "r+": read-write

```python
message = open("messages.txt", "w")
# 以写模式打开
message.write("123")
# 写入内容
message = open("messages.txt")
print(message.read())
message.close()
```

## 3. Saving variables with the shelve module

- Saving variables in a binary shelf file
  - A shelf file is similar to a dictionary in that it stores data as key-value pairs.

```python
import shelve

file = shelve.open("mydata")
items = ["A", "B", "C", "D", "E", "F"]
file['items'] = items
file.close()

file = shelve.open("mydata")
print(list(file.keys()))
print(list(file.values()))
print(file["items"])
file.close()
```
