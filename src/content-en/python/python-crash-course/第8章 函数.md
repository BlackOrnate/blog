### 2.1 Location

```python
def ABC(message1, message2):
    print(message1)
    print(message2)

ABC(1, 2)
```

### 2.2 Keyword parameters

```python
def ABC(message1, message2):
    print(message1)
    print(message2)

ABC(message2=1, message1=2)
```

### 2.3 Default values

```python
def ABC(message1, message2=2):
    print(message1)
    print(message2)

ABC(1)
```

## 4. Pass the list

### Prohibit functions from modifying the list

```python
def ABC(list):
    list[0] = 10
    return list

list = [1, 2, 3]
print(list)
print(ABC(list[:]))
print(list)
```
