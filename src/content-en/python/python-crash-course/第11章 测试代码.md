### 1.1 Unit Tests and Test Cases

- The unnittest module: code testing
- **unit test**: verifying that some aspect of a function is okay
- **test case**: a set of unit tests
- **full-coverage** test cases: a set of unit tests that cover every possible way a function can be used

### 1.2 Passable tests

```python
import unittest

def ABC(first, last):
    print(f"{first}: {last}")
    return first

class Test(unittest.TestCase):
    def test_case(self):
        message = ABC("a", "b")
        self.assertEqual("a", message)

if __name__ == "__main__":
    unittest.main()
```.

- .assertEqual(): asserts that actual and predicted values are equal
  - Recommended predicted value comes first, actual value comes second
- \_\_ name \_\_ : when used by itself it is \_\_\_ main \_\_, when it is called as a module it is its own name.

### 2.3 A class to test

```python
import unittest

class Animal:
    def __init__(self, name):
        self.name = name

    def show_name(self):
        print("Name: " + self.name)
        return self.name

class TestAnimal(unittest.TestCase):
    def test_get_name(self):
        self.assertEqual("ABC", Animal("abc").show_name())

if __name__ == "__main__":
    unittest.main()
