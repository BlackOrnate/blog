## 3. strings

## 3.1 Modifying the case of strings using methods

#### Capitalize the first character (.title())

```python
message = "error"
print(message.title())
# Error
```

#### Capitalize the entire string (.upper())

```python
message = "Error"
print(message.upper())
# ERROR
```

#### Make strings all lowercase (.lower())

```python
message = "Error"
print(message.lower())
# error
```

### 3.2 Using variables in strings

```python
ABC = "ABC"
abc = "abc"

message = ABC + abc
print(message)
# ABCabc

message = f"{ABC} {abc}"
print(message)
# ABC abc
# 相当于调用format()方法
# message = "{} {}".format(ABC, abc)
```

### 3.3 Adding whitespace with tabs or newlines

#### Tabs ("\t")

```python
message = "abc"
print("\t" + message)
# 	abc
```

#### Line feed character ("\n")

```python
message = "abc"
print("\n" + message)
# 
# abc
```

### 3.4 Remove whitespace at string endpoints (.strip())

```python
message = "  abc\n"
print(message.rstrip())
#   abc
print(message.lstrip())
# abc
#
print(message.strip())
# abc
```

### 3.5 Splitting a string (.split())

- split(): defaults to whitespace.

```python
message = "abcabc"
print(message.split("b", 1))
```
