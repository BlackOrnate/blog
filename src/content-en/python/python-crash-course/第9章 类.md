## 1. Creating and using classes

### 1.1 Creating Classes

```python
class Animal:
    def __init__(self, name="animal"):
        self.name = name

    def eat(self):
        print(self.name + " eat")
```

- init(): constructor, need to be created manually.

### 1.2 Creating instances based on classes

```
animal = Animal("ABC")
```

#### 1.2.1 Accessing Attributes

```python
print(animal.name)
```

#### 1.2.2 Calling methods

```
animal.eat()
```

## 2. Inheritance

### 2.1 Subclass methods \_\_ init \_\_()

```python
class Animal:
    def __init__(self, name):
        self.name = name

    def Eat(self):
        print(self.name + ": eat")

class Cat(Animal):
    def __init__(self, name, age):
        super().__init__(name)
        self.name = name
        self.age = age

    def Age(self):
        print("age: " + str(self.age))
```

- Since it is not possible to create constructor methods automatically, references to parent constructor methods in the constructor methods of subclasses also need to be manually rewritten

### 2.2 Overriding parent class methods

```python
class Animal:
    def __init__(self, name):
        self.name = name

    def Eat(self):
        print(self.name + ": eat")

class Cat(Animal):
    def __init__(self, name, age):
        super().__init__(name)
        self.name = name
        self.age = age

    def Eat(self):
        print(self.name + ": eating")
```

## 3. Importing classes

### 3.1 Importing a single class

- Animal.py

```python
class Animal:
    def __init__(self, name):
        self.name = name

    def Eat(self):
        print(self.name + ": eat")
```

- Main.py

```python
from animal import Animal

animal = Animal("ABC")

animal.Eat()
```

### 3.2 Importing multiple classes

- Animal.py

```python
class Animal:
    def __init__(self, name):
        self.name = name

    def Eat(self):
        print(self.name + ": eat")

class Cat(Animal):
    def __init__(self, name, age):
        super().__init__(name)
        self.name = name
        self.age = age

    def Eat(self):
        print(self.name + ": eating")
```

- Main.py

```python
from Animal import Animal, Cat

animal = Animal("ABC")
animal.Eat()

cat = Cat("abc", 1)
cat.Eat()
```

### 3.3 Importing an entire module

- Animal.py

```python
class Animal:
    def __init__(self, name):
        self.name = name

    def Eat(self):
        print(self.name + ": eat")

class Cat(Animal):
    def __init__(self, name, age):
        super().__init__(name)
        self.name = name
        self.age = age

    def Eat(self):
        print(self.name + ": eating")
```

- Main.py

```python
import Animal

animal = Animal.Animal("ABC")
animal.Eat()

cat = Animal.Cat("abc", 1)
cat.Eat()
```

### 3.4 Using Aliases

```python
from Animal import Cat as C

cat = C("abc", 1)
cat.Eat()
```
