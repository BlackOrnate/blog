## 1. What a list is

- A list is an array.

```python
ABC = ["A", "B", "C"]
print(ABC)
```

### 1.1 Accessing list elements

```python
ABC = ["A", "B", "C"]
print(ABC[0])
```

### 1.2 Indexing starts at 0 instead of 1

- Indexes can be negative
- Negative index means lookup from back to front.

```python
ABC = ["A", "B", "C"]
print(ABC[-1])
```

## 2. Modifying, adding, and removing elements

## 2.1 Modifying list elements

```python
ABC = ["A", "B", "C"]
ABC[0] = "a"
print(ABC)
```

### 2.2 Adding elements to a list

#### 2.2.1 Adding elements to the end of a list (.append())

```python
ABC = ["A", "B", "C"]
ABC.append("D")
print(ABC)
```

#### 2.2.2 Inserting elements into a list (.insert())

- Pre-insert at the selected subscript

```python
ABC = ["A", "B", "C"]
ABC.insert(0, "D")
print(ABC)
```

### 2.3 Removing elements from a list

#### 2.3.1 Removing Elements Using the del Statement

```python
ABC = ["A", "B", "C"]
del ABC[0]
print(ABC)
```

#### 2.3.2 Deleting elements using the pop() method (.pop())

- Removes the element with the specified subscript
- No subscript is specified, defaults to the end element

```python
ABC = ["A", "B", "C"]
ABC.pop(1)
print(ABC)
```

#### 2.3.3 Removing an element by value (.remove())

- Only the first specified value is removed

```python
ABC = ["B", "A", "B", "C"]
ABC.remove("B")
print(ABC)
```

## 3. Organizing lists

### 3.1 Permanently sorting a list using the sort() method (.sort())

- Lists consisting of letters only: sort by the ANSCII code of the first letter, from smallest to largest
- List consists of numbers only: sort by number size from smallest to largest.
- Mixed lists cannot be sorted

```python
ABC = ["B", "A", "B", "C"]
ABC.sort()
print(ABC)
```

- It can be changed to reverse by passing reverse=True.

```python
ABC = ["B", "A", "B", "C"]
ABC.sort(reverse=True)
print(ABC)
```

### 3.2 Temporary sorting of a list using the sort() function (.sorted())

```python
ABC = ["B", "A", "B", "C"]
print(ABC.sorted())
print(ABC)
```

### 3.3 Reversing the order of list elements (.reverse())

```python
ABC = ["B", "A", "B", "C"]
print(ABC.reverse())
```

### 3.4 Getting the length of a list (len())

```python
ABC = ["B", "A", "B", "C"]
print(len(ABC))
```
