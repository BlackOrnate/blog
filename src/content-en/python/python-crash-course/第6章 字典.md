## 1. Using Dictionaries

- Key-value pairs

```
AB = {"A": 1, "B": 2}
```

### 1.1 Accessing values in a dictionary

```python
AB = {"A": 1, "B": 2}
print(AB["A"])
```

### 1.2 Adding key-value pairs

```python
AB = {"A": 1, "B": 2}
AB["C"] = 3
print(AB)
```

### 1.3 Creating an empty dictionary

```python
AB = {}
print(AB)
```

### 1.4 Modifying values in the dictionary

```python
AB = {"A": 1, "B": 2}
AB["A"] = 3
print(AB)
```

### 1.5 Deleting key-value pairs

```python
AB = {"A": 1, "B": 2}
del AB["A"]
print(AB)
```

### 1.6 Using get() to access values

```python
AB = {"A": 1, "B": 2}
print(AB.get("C", "Not Find"))
```

## 2. Traversing the dictionary

## 2.1 Traversing all key-value pairs

- .items() gets the list of key-value pairs of the dictionary.

```python
AB = {"A": 1, "B": 2}
for key, value in AB.items():
    print("%s: %s" % (key, value))
```

### 2.2 Iterating over all keys in a dictionary

- .keys() gets the list of keys in the dictionary

```python
AB = {"A": 1, "B": 2}
for key in AB.keys():
    print(key)
```

### 2.3 Iterating over all values in a dictionary

- set(): converts an array into a set.
  - Elements in a set have no subscripts and cannot be repeated

```python
AB = {"A": 1, "B": 2}
for value in set(AB.values()):
    print(value)
```

## 3. nesting

### 3.1 Dictionary Lists

```python
AB = {"A": 1, "B": 2}
ab = {"a": 1, "b": 2}
ABab = [AB, ab]
print(ABab)
```

### 3.2 Storing lists in a dictionary

```python
AB = {"A": 1, "B": ["a", "b"]}
print(AB)
```

### 3.3 Storing a dictionary in a dictionary

```python
AB = {"A": {"a": 1}, "B": {"b": 2}}
print(AB)
```
