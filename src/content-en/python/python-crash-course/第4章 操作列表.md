## 1. Iterate over the entire list

- Enhanced for traversal

```python
ABC = ["A", "B", "C"]
for i in ABC:
    print(i)
# i不会因为for循环结束而销毁
# i会保留最后一次被赋值的元素值
```

## 2. Avoid indentation errors

### 2.1 Forgetting to indent

```python
ABC = ["A", "B", "C"]
for i in ABC:
print(i)
```

### 2.2 Forgetting to indent extra lines of code

```python
ABC = ["A", "B", "C"]
for i in ABC:
    print("~~~")
print(i)
```

### 2.3 Unnecessary indentation

```python
ABC = ["A", "B", "C"]
	print(ABC)
```

### 2.4 Unnecessary indentation after loops

```python
ABC = ["A", "B", "C"]
for i in ABC:
    print(i)

    print("Finish")
```

### 2.5 Missing colons

```python
ABC = ["A", "B", "C"]
for i in ABC
    print(i)
```

## 3. Creating a list of values

### 3.1 Creating a list of values with range()

- First argument: the lower limit of the list element values
- Second parameter: the upper limit of the list element value
- The third parameter is the step size, which is 1 by default.

```python
for value in range(1, 10, 2):
    print(value)
```

### 3.2 Simple statistical calculations on lists of numbers

```python
values = [1, 2, 3, 4, 5, 6, 7]
print(min(values))
print(max(values))
print(sum(values))
```

## 4. Using parts of a list

## 4.1 Slicing

- First parameter: start index, if not written then start from the table header
- Second parameter: terminating index, without writing the terminating index is ignored
- Slices the list from the start index to the end index (without the end index).

```python
values = [1, 2, 3, 4, 5, 6, 7]
print(values[1:5])
```

### 4.2 Copying a list

```python
values = [1, 2, 3, 4, 5, 6, 7]
a = values[:]
values[0] = 123
print(values)
print(a)
# 不能将value直接赋值给a
# 这样操作仅仅是指针传递
```

## 5. Tuples

- Tuples are similar to lists, but tuples cannot be modified.
- Neither the number of elements nor the values of the elements can be modified.

## 5.1 Defining tuples

```python
values = (1, 2, 3, 4, 5, 6, 7)
print(values[0])
```

### 5.2 Iterating over all values in a tuple

```python
values = (1, 2, 3, 4, 5, 6, 7)
for value in values:
    print(value)
```

### 5.3 Modifying tuple variables

```
values = (1, 2, 3, 4, 5, 6, 7)
values = (11, 12, 13, 14, 15, 16, 17)
```
