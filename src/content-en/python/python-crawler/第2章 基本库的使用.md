## 1. Using urllib

- Implement HTTP request sending, and do not need to care about the HTTP protocol itself or even the underlying implementation, just specify the request URL, request header, request body and other information.
- You can convert the response from the server into Python objects.

  - You can turn the response from the server into a Python object, which makes it easy to get information about the response.
    - Response status code
    - Response status code Response headers
    - Response body
    - The response body.
- The urllib library contains 4 modules

  - request: the most basic HTTP request module, which simulates the sending of a request.
  - error: exception handling module
  - parse: tool module
  - robotparser (less used): recognizes the robot.txt file of a website and determines whether the website is crawlable or not.

### 1.1 Sending a Request

#### 1.1.1 urlopen

- Grab the source code of a web page.
- Syntax:

  ```python
  urllib.request.urlopen(url, 
                         data=None, 
                         [timeout, ]*, 
                         cafile=None, 
                         capath=None, 
                         cadefault=False, 
                         context=None)
  ```

```python
import urllib.request

response = urllib.request.urlopen("https://www.python.org/")

print(response.read().decode("utf-8"))
# 获取网页源码
print(type(response))
# <class 'http.client.HTTPResponse'>
print(response.status)
# 获取网页状态码
print(response.getheaders())
# 获取网页响应头
print(response.getheader("Server"))
# 获取网页指定的响应头
```

##### data Parameters

- Parameters need to be converted to bytes
- When parameters are passed in, the request method will be converted from GET to POST.

```python
import urllib.request
import urllib.parse

data = bytes(urllib.parse.urlencode({"name": "abc"}), encoding='utf-8')
response = urllib.request.urlopen("https://www.httpbin.org/post", data=data)
print(response.read().decode('utf-8'))
# 模拟表单提交，以 POST 方式传输数据
```

##### timeout parameter

- Sets the timeout period.
- The unit is seconds.

```python
import socket
import urllib.request
import urllib.error

try:
    response = urllib.request.urlopen(
        "https://www.httpbin.org/get", timeout=0.1)
except urllib.error.URLError as e:
    if isinstance(e.reason, socket.timeout):
        print("TIME OUT")
```

##### Other parameters

- The context parameter:
  - Parameters must be of type ssl.
  - SSLContext class Used to specify SSL settings
- cafile parameter:
  - Used to specify the CA certificate
- capath parameter:
  - Used to specify the path of CA integer
- cadefault parameter (deprecated)

#### 1.1.2 Request

- Separate request into a single object
- Richer and more flexible configuration of parameters

```python
import urllib.request

url = "https://www.python.org/"
request = urllib.request.Request(url)
response = urllib.request.urlopen(request)
print(response.read().decode("utf-8"))
```

- Syntax:

  ```python
  urllib.request.Request(url, 
                         data=None, 
                         headers={}, 
                         origin_req_host=None, 
                         unverifiable=False, 
                         method=None)
  ```

  - url:

    - Used to request a URL
  - data:

    - Parameters need to be of type byte
    - The dictionary needs to be qualified with urllib.parse.urlencode() first.
  - headers:

    - Request headers
    - is a dictionary
    - When constructing a request, this can be constructed directly from the headers parameter, or added by calling the add\_header method of the request instance.
    - Disguise the browser by modifying the User-Agent.
  - origin\_req\_host:

    - The host name or IP address of the requesting party.
  - unverifiable:

    - Used to specify whether the request is unverifiable
    - The user does not have sufficient privileges to receive the result of this request.
  - method:

    - Used to specify the method to be used for the request
      - GET
      - POST
      - PUT

```python
from urllib import request, parse

url = "https://www.httpbin.org//post"
headers = {
    "User-Agent": "Edg/105.0.1343.42",
    "Host": "www.python.org"
}
user = {"name": "abc"}
data = bytes(parse.urlencode(user), encoding="utf-8")

req = request.Request(url=url, data=data, headers=headers, method="POST")
# req = request.Request(url=url, data=data, method="POST")
# req.add_header("User-Agent", "Edg/105.0.1343.42")

response = request.urlopen(req)

print(response.read().decode("utf-8"))
```

- Use add\_header() to add a response header

```python
from urllib import request, parse

url = "https://www.httpbin.org//post"
user = {"name": "abc"}
data = bytes(parse.urlencode(user), encoding="utf-8")

req = request.Request(url=url, data=data, method="POST")
req.add_header("User-Agent", "Edg/105.0.1343.42")
req.add_header("Host", "www.python.org")

response = request.urlopen(req)

print(response.read().decode("utf-8"))
```

#### 1.1.3 Advanced Usage

- Handler: various handlers

  - Handling login authentication
  - Handling Cookies
  - Handling proxy settings
  ----
- The BaseHandler class in urllib.request is the parent class of all other Handler classes.

  - Provides the most basic methods
- Examples of subclasses

  - HTTPDefaultErrorHandler: used to handle HTTP response errors, all errors will throw an exception of type HTTPError.
  - HTTPRedirectHandler: used to handle redirection.
  - HTTPCookieProcesser: Used to process cookies.
  - ProxyHandler: used to set the proxy, default is empty.
  - HTTPPasswordMgr: used to manage passwords.
  - HTTPBasicAuthHandler: used to manage authentication.
- OpenerDirector (Opener): urlopen method is an Opener provided by urllib.
- Use Handler class to build Opener class

##### Authentication

- Authentication is done with the HTTPBasicAuthHandler module.

```python
from urllib.request import HTTPBasicAuthHandler, HTTPPasswordMgrWithDefaultRealm, build_opener
from urllib.error import URLError

import ssl
ssl._create_default_https_context = ssl._create_unverified_context
# 如果没有进行SSL认证，可以通过此方式将SSL认证全局关闭

username = "admin"
password = "admin"
url = "https://ssr3.scrape.center/"

p = HTTPPasswordMgrWithDefaultRealm()
p.add_password(None, url, username, password)
# 将账号密码封装
handler = HTTPBasicAuthHandler(p)
opener = build_opener(handler)

try:
    result = opener.open(url)
    html = result.read().decode("utf-8")
    print(html)
except URLError as e:
    print(e.reason)
```

##### proxy

- Add Proxy

```python
from urllib.request import ProxyHandler, build_opener
from urllib.error import URLError

    "http": "http://127.0.0.1:8080",
    "https": "https://127.0.0.1:8080"

##### Cookie

- Get Cookie

```python
from http.cookiejar import CookieJar
from urllib.request import HTTPCookieProcessor, build_opener

cookie = CookieJar()
handler = HTTPCookieProcessor(cookie)
opener = build_opener(handler)

response = opener.open("https://www.baidu.com/")

for item in cookie:
    print(f"{item.name} = {item.value}")
```

- Storing Cookies
  - MozillaCookieJar: for handling events related to cookies and files
    - Read Cookie
    - Save Cookie
    ---

from http.cookiejar import MozillaCookieJar

### 1.2 Handling exceptions (error)

#### 1.2.1 URLError

- error Base class of the exception module
- Returns the cause of an error.

```python
from urllib import request, error

try:
    request = request.urlopen("https://www.bilibili.com/")
except error.URLError as e:
    print(e.reason)
```

#### 1.2.2 HTTPError

- Specialized for handling HTTP request errors
  - Authentication request failed
  ---
- Possesses the following attributes
  - code: status code
  - reason: the reason for the error
  - headers: request headers

```python
from urllib import request, error

try:
    request = request.urlopen("https://www.bilibili.com/")
except error.HTTPError as e:
    print(f"{e.code}\n{e.reason}\n{e.headers}")
```

- reason is not necessarily a string, it could be an object.

```python
from urllib.request import urlopen
from urllib.error import URLError
from socket import timeout

try:
    response = urlopen("https://www.bilibili.com/", timeout=0.01)
except URLError as e:
    if isinstance(e.reason, timeout):
        print("Time out")
```

### 1.3 Parsing links (parse)

#### 1.3.1 urlparse

- Used for URL identification and segmentation
- Return value type: ParseResult (which is a tuple)
  - attribute:
    - scheme: protocol name
    - netloc: domain name
    - path: access path
    - params: parameters
    - query: query condition
    - fragment: anchor point
  - Content can be retrieved by attribute name
  - The content can also be retrieved by subscript

```python
from urllib.parse import urlparse

result = urlparse("https://www.baidu.com/index.html;user?id=5#comment")
print(type(result))
print(result)
```.

- Syntax

  ```python
  urlparse(url, scheme='', allow_fragments=True)
  ```

  - url: the URL to be parsed
  - scheme: use this scheme when the URL does not specify a protocol
  - allow\_fragments: if or not allow anchors to be logged.
    - If not, attribute the anchor to the last attribute in the URL.

#### 1.3.2 urlunparse

- Used to construct URLs
- The length of the received parameter must be 6
  - As long as it's 6, it's fine.
    - Arrays
    - Arrays
    ---

#### 1.3.5 urljoin

- Generate a link
- Syntax

  ```python
  urljoin(base, url, allow_fragments=True)
  ```

  - base: base link
  - url: new link
- The urljoin method analyzes the base's scheme, netloc, and path to add the missing parts of the url
- Normal

_```python
from urllib.parse import urljoin

print(urljoin("https://www.baidu.com?wd=abc", "https://www.bilibili.com/index.html"))
# https://www.bilibili.com/index.html
print(urljoin("https://www.baidu.com?wd=abc", "https:///"))
# https://www.baidu.com/
print(urljoin("https://www.baidu.com?wd=abc", "https:///index.html"))
# https://www.baidu.com/index.html
```

- Non-human

```python
from urllib.parse import urljoin

print(urljoin("https://www.baidu.com?wd=abc", "https"))
# https://www.baidu.com/https
print(urljoin("https://www.baidu.com?wd=abc", "https:"))
# https://www.baidu.com?wd=abc
print(urljoin("https://www.baidu.com?wd=abc", "https://"))
# https://www.baidu.com?wd=abc

print(urljoin("https://www.baidu.com?wd=abc", "www.bilibili.com/index.html"))
# https://www.baidu.com/www.bilibili.com/index.html
print(urljoin("https://www.baidu.com?wd=abc", "//www.bilibili.com/index.html"))
# https://www.bilibili.com/index.html

print(urljoin("https://www.baidu.com?wd=abc", "?ab=123"))
# https://www.baidu.com?ab=123
print(urljoin("https://www.baidu.com?wd=abc", "#123"))
# https://www.baidu.com?wd=abc#123
```

### 1.4 Analyzing the Robots Protocol (robotparser)

#### 1.4.1 Robots protocol

- Exclusion criteria for web crawlers
- Usually a robots.txt text file (stored in the root directory of a website).
- Exploration crawlers first check for the existence of this file when visiting a site

  - Exists: it crawls according to the crawl scope defined in it
  - Does not exist: the search crawler will visit all pages that can be accessed directly
- Examples

  ```
  User-agent: *
  Disallow: /
  Allow: /public/
  ```

  - User-agent: name of the search crawler
    - \*: all
  - Disallow: directories that are not allowed to be crawled
    - /: All
  - Allow: directories that are allowed to be crawled (usually used with Disallow)

#### 1.4.2 robotparser

- According to robots.txt to determine whether the crawler has the permission to crawl this web page

```python
from urllib.robotparser import RobotFileParser

rp = RobotFileParser()
rp.set_url("https://www.baidu.com/robots.txt")
rp.read()

print(rp.can_fetch("Baiduspider", "https://www.baidu.com"))
print(rp.can_fetch("Baiduspider", "https://www.baidu.com/homepage/"))
print(rp.can_fetch("Googlebot", "https://www.baidu.com/homepage/"))
```

- You can also use pars to parse robots.txt.

```python
from urllib.request import urlopen
from urllib.robotparser import RobotFileParser

rp = RobotFileParser()
rp.parse(urlopen("https://www.baidu.com/robots.txt").read().decode("utf-8").split("\n"))

print(rp.can_fetch("Baiduspider", "https://www.baidu.com"))
print(rp.can_fetch("Baiduspider", "https://www.baidu.com/homepage/"))
print(rp.can_fetch("Googlebot", "https://www.baidu.com/homepage/"))
```

### 3.2 match

- Match from the beginning of the string, if the beginning does not match, the whole match fails.

#### 3.2.1 Match target

```python
from re import match

content = "hello 12345 world 12345"
result = match("^hello\\s(\\d*)\\sworld", content)

print(result)
print(result.group())
print(result.group(1))
print(result.span())
```

#### 3.2.2 Generic Matching

- Matches all characters except newlines

```python
from re import match

content = "hello 12345 world 12345"
result = match("^hello.*world", content)

print(result)
print(result.group())
print(result.span())
```

#### 3.2.3 Greedy vs. non-greedy

- Greedy

```python
from re import match

content = "hello 12345 world 12345"
result = match("^hello.*(\\d+).*world", content)

print(result)
print(result.group(1))
```

- Non-greedy

```python
from re import match

content = "hello 12345 world 12345"
result = match("^hello.*?(\\d+).*world", content)

print(result)
print(result.group(1))
```

- Note that
  - If the match is at the end of a string, then the non-greedy pattern may not match the content

```python
from re import match

content = "hello 12345 world 12345"

result1 = match("^hello.*?world(.*)", content)
print(result1.group(1))

result2 = match("^hello.*?world(.*?)", content)
print(result2.group(1))
```

#### 3.2.4 Modifiers

| Modifiers | Description |
| --- | --- |
| re.I | Match is case insensitive | re.
| re.L | Implement localization-aware matching | | re.M | Multi-case matching | | re.
| re.M | Multi-line matching (affects ^ and $) |
| M | Multi-line matching (affects ^ and $) | re.
| re.U | Parses characters according to Unicode character set (affects \w, \W, \b and \B) | re.
| re.X | Make regular expressions easier to understand | | re.

#### 3.2.5 Escape Matching

```python
from re import match

content = "hello (12345) world 12345"

result = match("^hello.*?\\(\\d*\\).*?world", content)
print(result)
```

        detail_url = urljoin(BASE_URL, item)
