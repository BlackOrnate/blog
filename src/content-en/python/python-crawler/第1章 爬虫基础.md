## 1. HTTP Fundamentals

### 1.1 URIs and URLs

- **URI**: Uniform Resource Identifier

  - **URL**: Uniform Resource Locator

    - ```
      http://localhost:8080/Study/test.jsp
      ```.
    - Specifies a unique way to access the resource (access protocol, access path, and resource name)
  - **URN**: Uniform Resource Name

    - ```
      urn:id:123456
      ```.
    - Names the resource only, but does not specify how to locate it
- The basic composition format of a URL

  ```
  scheme://[username:password@]hostname[:port][/path][;parameters][?query][#fragment]
  ```

  - **scheme(protocol)**: protocol

    - Common protocols: hrrp, https, ftp, etc.
  - **username, password**: username, password

    - In some cases, the URL requires a username and password to access it.
  - **hostname**: host address

    - Can be a domain name or IP address
  - **port**: port

    - ```
      http://localhost:8080
