## 1. Object概述

- 超类、基类，所有类的直接或间接父类，位于继承树的最顶层
- 任何类，如果没有书写extends显示继承某个类，都默认直接继承Object类，否则为间接继承
- Object类中所定义的方法，是所有对象都具备的方法
- Object类型可以存储任何对象
  - 作为参数，可接受任何对象
  - 作为返回值，可返回任何对象

## 2. getClass方法

- public final Class<?> getClass(){}
- 返回引用中存储的实际对象类型
- 应用：通常用于判断两个引用中实际存储对象类型是否一致

```java
public class Test {
    public static void main(String[] args) {
        Student student1 = new Student("student1", 10);
        Student student2 = new Student("student2", 20);

        // 判断student1和student2是否为同一类型
        Class class1 = student1.getClass();
        Class class2 = student2.getClass();
        System.out.println(class1 == class2);
    }
}
```

```java
public class Student {
    private String name;
    private int age;

    public Student() {
    }

    public Student(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        this.age = age;
    }
}
```

## 3. hashCode方法

- public int hashCode(){}
- 返回该对象的哈希码值
- 哈希值根据**对象的地址**或**字符串**或**数字**使用hash算法计算出来的int类型的数值
- 一般情况下相同对象返回相同哈希码

```java
public class Test {
    public static void main(String[] args) {
        Student student1 = new Student("student1", 10);
        Student student2 = new Student("student2", 20);

        System.out.println(student1.hashCode());
        System.out.println(student2.hashCode());
    }
}
```

## 4. toString方法

- public String toString()){}
- 返回该对象的字符串表示（表现形式）
- 可以根据程序需求覆盖该方法，如：展示对象各个属性值

```java
public class Test {
    public static void main(String[] args) {
        Student student1 = new Student("student1", 10);
        Student student2 = new Student("student2", 20);

        System.out.println(student1.toString());
        System.out.println(student2.toString());
    }
}
```

```java
public class Student {
    private String name;
    private int age;

    public Student() {
    }

    public Student(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        this.age = age;
    }

    @Override
    public String toString() {
        return "Student{" +
                "name='" + name + '\'' +
                ", age=" + age +
                '}';
    }
}
```

## 5. equals方法

- public boolean equals(Object obj){}
- 默认实现为(this == obj)，比较两个对象地址是否相同
- 可进行覆盖，比较两个对象的内容是否相同

```java
public class Test {
    public static void main(String[] args) {
        Student student1 = new Student("student1", 10);
        Student student2 = new Student("student2", 20);
        Student student3 = new Student("student1", 10);

        System.out.println(student1.equals(student2));
        System.out.println(student1.equals(student3));
    }
}
```

```java
public class Student {
    private String name;
    private int age;

    public Student() {
    }

    public Student(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        this.age = age;
    }

    @Override
    public boolean equals(Object o) {
        if (this == o) return true;
        if (!(o instanceof Student)) return false;
        Student student = (Student) o;
        return age == student.age && name.equals(student.name);
    }
}
```

## 6. finalize方法

- 当对象被判定为垃圾对象时，由JVM自动调用此方法，用以标记垃圾对象，进入回收队列
- 垃圾对象：没有有效引用指向此对象时，为垃圾对象
- 垃圾回收：由GC销毁垃圾对象，释放数据存储空间
- 自动回收机制：JVM的内存耗尽，一次性回收所有垃圾对象
- 手动回收机制：使用 System. gc(); 通知JVM执行垃圾回收
- 此种方法已被JDK删除

```java
public class Test {
    public static void main(String[] args) {
        Student student1 = new Student("student1", 10);
        Student student2 = new Student("student2", 20);
        new Student("student3", 30);
        new Student("student4", 40);

        System.gc();
    }
}
```

```java
public class Student {
    private String name;
    private int age;

    public Student() {
    }

        this.name = name;
```
