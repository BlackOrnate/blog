## 1. Java帝国的诞生

- C & C++
  - 1972年C诞生
    - 贴近硬件，运行极快，效率极高
    - 操作系统，编译器，数据库，网络系统等
    - 指针和内存管理
  - 1982年C++诞生
    - 面向对象
    - 兼容C
    - 图形领域、游戏等
- Java初生
  - 1995年的网页简单而粗糙，缺乏互动性
  - 图形界面的程序（Applet）
  - Java 2 标准版：桌面
  - Java 2 移动版：手机
  - Java 2 企业版：服务器
- Java发展
  - 基于Java开发了许多的平台，系统，工具
    - 构建工具：Ant，Maven，Jekins
    - 应用服务器：Tmocat，Jetty，Jboss，Websphere，weblogic
    - Web开发：Struts，Spring，Hibernate，myBatels
    - 开发工具：Eclipse，Netbean，intellij idea，Jbuilder
  - 2006：Hadoop（大数据领域）
  - 2008：Android（手机端）

- 高性能
- 分布式
- 动态性
- 多线程

## 3. Java三大版本

- Write Once、Run Anywhere
- JavaSE：标准版（桌面程序，控制台开发）
- JavaME：嵌入式开发（手机，小家电）
- JavaEE：E企业级开发（web端，服务器开发）

## 4. JDK JRE JVM

- JDK：Java Development Kit
- JRE：Java Runtime Environment
- JVM：Java Virtual Machine
